# MicSplitter - マイク音声分配ツール - 開発背景

## 開発の背景

近年は、WEB通話とAI音声入力ツールの普及により、1つのマイク入力を複数のアプリケーションに振り分けたい機会が増えてきました。しかし、Windows標準の機能ではマイク入力を分配することはできません。

この他に、リアルタイムボイスチェンジャーや録音アプリを使用したい場合にも、物理マイクの音をそのまま相手に届けたくない場面が存在します。

そこで、私はマイク入力を複数のアプリケーションに分配するツール「MicSplitter」を開発することにしました。どちらの仮想マイクに音声を届けるかはツール上で切り替えることができます。

アプリごとに異なる仮想マイクを割り当てることで、ユーザーがキーボード等の操作により、配信先に届ける音声を切り替えることができます。

## 類似ツール

本ツールが目的としていることは、VB-Audio社の「Voicemeeter」と「VB-Audio Virtual Cable」を組み合わせることで実現できます。しかし、Voicemeeterは非常に高機能な反面、多機能であるがゆえに操作が複雑で設定難易度がとても高くなっています。また、多機能であるため処理が重くなる傾向があります。

## 前提ツール

本ツールを実行するためには仮想マイクが必要となりますが、デバイスドライバを自作するためには専門的な高い知識と高額なEV証明書の取得などが必要です。そのため、デバイスドライバについては有名なフリーソフトであるVB-Audio社の「Virtual Cable」を使用します。 https://vb-audio.com/Cable/
